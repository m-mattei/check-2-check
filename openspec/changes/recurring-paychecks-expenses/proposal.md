# Recurring Paychecks and Expenses

## Problem Statement
Users currently need to manually create paychecks and expenses for each occurrence. This is tedious for regular, predictable transactions like:
- Bi-weekly paychecks
- Monthly rent/mortgage payments
- Weekly subscriptions
- Quarterly insurance payments

## Proposed Solution
Add recurring transaction support with configurable frequency patterns including:
- Daily
- Weekly (with day of week selection)
- Bi-weekly
- Monthly (with day of month selection)
- Quarterly
- Annually

## Benefits
- Reduce manual data entry
- Improve budget accuracy with automatic future transaction generation
- Better financial planning visibility
