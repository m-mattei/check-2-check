## ADDED Requirements

### Requirement: Theme Color Consistency (Secondary Color)
The system SHALL use the theme's `secondary` color scheme for highlighting selected or active items, such as the current selection on the budget calendar.

#### Scenario: Selected Date Styling
- **WHEN** the user selects a date on the calendar
- **THEN** the system SHALL apply a background decoration using the `secondary` color from the current `ColorScheme`
